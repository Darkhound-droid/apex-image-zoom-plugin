# Contributing

Thanks for your interest in improving this project. Contributions are very welcome.

## Report Bugs
If you find a bug, please open a GitHub Issue with:
- What happened
- Steps to reproduce it
- Your Oracle APEX version and browser details

## Suggest Features
Have an idea to improve the plug-in? Open a GitHub Issue and describe:
- The problem you want to solve
- Your proposed solution
- Any usage example

## Submit a Pull Request
1. Fork the repository and create a feature branch
2. Make your changes and keep them focused
3. Add a clear description of what changed and why
4. Open a pull request for review

Thank you for contributing and helping make this plug-in better for everyone.
